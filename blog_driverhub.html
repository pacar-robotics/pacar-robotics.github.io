The REV Driver Hub

The REV Driver Hub is the newest piece of the REV Control System, designed to get rid of the hassle of dealing with the phones and producing a more streamlined driver experience. Thanks to the many USB ports, you no longer need a separate USB Hub to connect both controllers. Even the charging is easier, with double A batteries that can be changed in a pinch.

The Driver Hub OS

One of the problems we had with the old phone control system was the constant notifications and software updates. One of our best practice matches that would have been a new personal record was ruined due to a software update notification that closed the app mid-match! Thanks to the new dedicated Driver Hub OS, we no longer need to worry about software updates ruining our games or having to sign into Google.

Since the Driver Hub runs Android OS, it can run any app you wish to put on it. If you download the apk of an app and use a File Manager App to install it, you can get any app you like on it!
